Getting Started

  1. Make sure you have a high quality image saved in your img folder inside the day 2 folder from yesterday. Unsplash is a great place to look for photos.
  2. Login to yourgithub account
  3. Make a new public repository in github. Name it "portfolio" or something similar. Check out todays slides if you are unsure how to make a repository.
  4. Open up git bash.
  5. Open the Day 2 Folder with VSCode.

Done?

  1. Type "I have everything set up and have made a git repository" in the chat.
  2. Check out the review slides and make sure you understand everything. Ask questions about anything you don't understand.
  3. Read ahead on today's slides Ask questions about anything you don't understand.
